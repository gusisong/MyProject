# 自驱动贪吃蛇游戏原型设计文档

## 1. 项目概述
本项目旨在创建一个基于 Web 的自驱动贪吃蛇游戏原型。游戏将展示一条由 AI（A* 算法）控制的小蛇，它会自动寻找并吃掉苹果以增长。游戏包含血量系统，碰到边界会扣血，咬到自己则游戏结束。

## 2. 视觉设计 (精致像素风)
*   **配色方案**:
    *   背景: 深灰蓝 (#1a1c2c)，带有 20x20 的暗色网格。
    *   蛇身: 翠绿色像素点 (#33e033)，头部有明显区分。
    *   苹果: 鲜红色像素点 (#ff0044)，带白色高光。
*   **UI 元素**:
    *   顶部血条: 20 个红色/灰色小方块表示 20 HP。
    *   得分显示: 当前长度或吃掉的苹果数。
*   **网格大小**: 400x400 像素的画布，每个单元格 20x20 像素。

## 3. 核心机制
*   **生命值系统**: 
    *   初始 20 HP。
    *   撞墙惩罚: 每次碰撞扣除 1 HP。
*   **死亡条件**:
    *   HP 归零。
    *   蛇头碰到自己的身体。
*   **自动增长**: 吃掉苹果后，蛇身长度增加 1。

## 4. AI 控制逻辑 (A* 算法)
*   **寻路模式**: 实时计算蛇头到苹果的最短避障路径。
*   **生存模式**: 若无通往苹果的路径，AI 将寻找能移动的最远空格（通常是追踪蛇尾），以最大化生存空间。

## 5. 技术实现
*   **技术栈**: HTML5 Canvas, Vanilla JavaScript (ES6+)。
*   **部署**: 单个 `index.html` 文件，包含所有样式和逻辑。
